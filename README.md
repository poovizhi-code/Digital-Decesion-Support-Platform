# 🌴 Digital Decision-Support Platform for Oil Palm Farmers

## 📖 Overview
The Digital Decision-Support Platform for Oil Palm Farmers is a web-based application designed to support farmers by providing essential agricultural information through a single platform. It helps farmers make informed decisions using real-time weather updates, government scheme information, yield prediction, and expert consultation.

## ✨ Features
- 🌦️ Live Weather Updates
- 📢 Government Schemes Information
- 🌱 Yield Prediction
- 👨‍🌾 Expert Consultation
- 🎤 Voice Assistance
- 🔐 User Registration and Login
- 📊 User Dashboard
- 🚪 Secure Logout

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeather API
- GitHub Pages
